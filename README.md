# 📊 Data Modeling & Data Engineering Fundamentals

##  Overview

This repository showcases my hands-on learning journey in **Data Modeling and Modern Data Engineering** using tools like **PySpark, Delta Lake, and Databricks**.

It covers end-to-end concepts from **data ingestion (Bronze layer)** to building a **dimensional data warehouse (Gold layer)** using best practices followed in industry.
